# NovelReader

**NovelReader**는 조아라, 노벨피아와 같은 웹 소설 **서비스 플랫폼**입니다.

## 프로젝트 STS에서 불러오기

1. project import
2. Maven 선택
3. Check out Maven Projects from SCM

## 프로젝트 IntelliJ에서 불러오기

1. git clone 후 IntelliJ로 프로젝트 열기
2. Tomcat 서버 설정
3. Project Structure에서 artifact 설정
4. Tomcat에 artifact 빌드하고 실행
